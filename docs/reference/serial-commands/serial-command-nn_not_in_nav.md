# <small>``<N Q|[«cmd» «value»]>``</small> <br/> List SensorCAM Defines or Execute Commands

Serial commands to List SensorCAM defines or execute **SensorCAM** commands.

## Commands

* ``<N>`` List current and alternate defines for SensorCam base vpins
* ``<N Q>`` Lists full set of SensorCam sensor states
* ``<N «cmd» «value»>`` Execute SensorCam command with parameters

## Parameters

* **cmd**: SensorCAM command. See the [SensorCAM manual](../../products/ex-sensorcam/ex-sensorcam.md) for more information.
* **value**: Prarmeters needed for the SensorCAM command. [SensorCAM manual](../../products/ex-sensorcam/ex-sensorcam.md) for more information.
