# <small>``</ FREEALL|[RESERVE|FREE «section»]>``</small> <br/> Reserve or Free Sections

Serial commands to reserve or free sections.

## Commands

* ``</ RESERVE «section»>`` Flag section as reserved
* ``</ FREE «section»>`` Free reserve on section
* ``</ FREEALL>`` Free all reserves

## Parameters

* **section**: Section to  free or reserve
