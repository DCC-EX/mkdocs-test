# Getting Started with DCC-EX

If you are just starting here for the first time, it's more than likely you are interested in the possibility of using a **DCC-EX** [EX-CommandStation](ex-commandstation/overview.md) to run some trains.

A Command Station is the electronic device that translates throttle commands, such as telling a loco to move, into electronic signals sent on the track to the loco.

These signals are usually DCC  (Digital Command Control) which allows commands that control multiple locos on the same piece of track.

However, the **DCC-EX** [EX-CommandStation](ex-commandstation/overview.md) is also capable of running DC (Direct Current) locos as there are millions of them still in use.

You have the option to buy a **DCC-EX** [EX-CommandStation](ex-commandstation/overview.md) [ready-to-run](../purchasing/official-sellers.md) or [build it yourself](../diy/diy.md) from cheaper parts.

You will probably want to install a throttle on your phone/tablet. You can get the recommended Engine Driver Android app on the Play Store: <br/> [![Play Store](/_static/images/engine-driver/download.png)](https://play.google.com/store/apps/details?id=jmri.enginedriver) <br/>
or check our [list of apps and devices](../throttles/throttles.md) that we know work with [EX-CommandStation](ex-commandstation/overview.md).
