# ThrottleCard

Take control of your model railroad with this elegantly designed app that brings vintage charm to modern model train control. Whether you're a seasoned engineer or just starting your journey in model railroading, **ThrottleCard** delivers powerful features wrapped in a beautiful, easy-to-use interface. You can connect ThrottleCard to JMRI and connect JMRI to your **EX-CommandStation**, or you if you are not going to use JMRI, you can connect directly to the Command Station if you install a WiFi board.

Please visit their [website](https://throttlecard.com)

You can find it in the App Store: [ThrottleCard](https://apps.apple.com/app/apple-store/id6739982126?pt=451685&ct=dcc_ex&mt=8)

## Features

* Control multiple trains simultaneously with intuitive multi-throttle support
* Beautiful vintage-inspired gauges and controls with realistic animations
* Game Controller Support! Easily drive your Trains with a PlayStation, Xbox, or Nintendo Switch Controller with pre-mapped buttons
* Easily create consists on the fly right from within the app
* Automatic command station discovery
* Comprehensive locomotive function control
* Full turnout control
* Haptic Feedback for intuitive responsiveness and operation
* Mirror-Mode mode for comfortable operation in either hand
* Multiple classic railroad-inspired themes
* iPad/macOS support
* Native DCC-EX protocol support, including CV programming

With many more exciting features in the roadmap!
